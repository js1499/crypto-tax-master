# Test Data Directory

This directory contains test CSV files for debugging and development purposes.

## Adding Your CSV File

1. Place your CSV file in this directory
2. Name it something descriptive like `tax-report-2024.csv` or `sample-transactions.csv`
3. Commit and push to GitHub:
   ```powershell
   git add test-data/your-file.csv
   git commit -m "Add test CSV file for debugging tax calculations"
   git push
   ```

## Privacy Note

⚠️ **Important**: CSV files may contain sensitive financial data. Consider:
- Using a private GitHub repository
- Redacting or anonymizing sensitive information before uploading
- Removing the file after debugging is complete
